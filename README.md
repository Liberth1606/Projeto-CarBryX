# 📚 CarBryX

Sistema web para gerenciamento de acervo de livros. Desenvolvido como projeto de estudo com foco em CRUD completo, separação de camadas e integração com frontend usando Thymeleaf.

## 🚀 Tecnologias utilizadas

- Java 21
- Spring Boot 3.5.14
- Spring Data JPA
- Spring Web
- Thymeleaf
- H2 Database
- Maven
- HTML5 / CSS3

## 📋 Funcionalidades

- Cadastrar novo livro
- Listar todos os livros
- Editar informações de um livro
- Excluir um livro
- Interface web responsiva e amigável

## Como executar
1. Abra no IntelliJ
2. Execute `CarBryXApplication.java`
3. Acesse `http://localhost:8080`
